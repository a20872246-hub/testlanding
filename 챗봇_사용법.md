# 🐕 강아지 AI 챗봇 - 빠른 시작 가이드

## ✅ 완료된 작업

우측 하단에 강아지 모양 아이콘의 AI 챗봇이 추가되었습니다!

- 🐕 **강아지 아이콘** 우측 하단 고정
- 🤖 **Gemini 2.5 Flash API** 연동
- 💬 **강아지 행동 교정 전문 상담** 제공
- ✨ **반응형 디자인** (모바일/PC)

## 🚀 바로 시작하기

### 1단계: API 키 받기

1. https://aistudio.google.com/app/apikey 접속
2. 구글 계정 로그인
3. "Create API Key" 클릭
4. API 키 복사

### 2단계: API 키 설정

프로젝트 폴더에서 `.env.local` 파일을 열고:

```
GEMINI_API_KEY=여기에_복사한_API_키_붙여넣기
```

### 3단계: 서버 재시작

터미널에서:
```bash
# Ctrl+C로 서버 중지
# 다시 실행
npm run dev
```

### 4단계: 테스트

브라우저에서 http://localhost:3000 접속 후 우측 하단 강아지 아이콘 클릭!

## 💬 테스트 질문 예시

챗봇에게 이렇게 질문해보세요:

1. "강아지가 계속 짖어서 이웃에게 민폐예요"
2. "분리불안이 심한데 어떻게 해야 하나요?"
3. "산책 중에 다른 강아지를 보면 공격적이에요"
4. "배변 훈련이 안 되는데 방법이 있나요?"

## 🎯 챗봇 특징

### AI가 답변하는 것 ✅
- 강아지 짖음, 공격성, 분리불안
- 배변 훈련, 산책 예절
- 문제 행동 교정 방법
- 훈련 팁과 실천 방법

### AI가 답변하지 않는 것 ❌
- 강아지 건강/질병 (→ 수의사 상담 권유)
- 강아지와 무관한 일반 질문
- 다른 동물 관련 질문

## 🎨 커스터마이징

### 색상 변경하고 싶다면
`components/DogChatbot.tsx` 파일에서:
- `purple-600` → `blue-600`, `red-600` 등으로 변경

### 환영 메시지 변경하고 싶다면
`components/DogChatbot.tsx` 파일에서:
```tsx
content: '원하는 첫 메시지로 변경하세요!'
```

## 🔧 문제 해결

### 챗봇이 안 보여요
→ 브라우저 새로고침 (Ctrl+R)

### "API 키가 설정되지 않았습니다" 에러
→ `.env.local` 파일 확인 후 서버 재시작

### 응답이 안 와요
→ 인터넷 연결 확인 & API 키가 유효한지 확인
