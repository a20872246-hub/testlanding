## 3. Problem 섹션

**파일: `components/sections/Problem.tsx`**

고객이 겪는 문제점을 나열합니다.

```yaml
섹션_제목: {문제 제기 제목}
섹션_설명: {문제 제기 설명}

문제_목록:
  - 아이콘: {아이콘명}
    제목: {문제 제목}
    설명: {문제 설명}
```

**현재 예시:**
```yaml
섹션_제목: 이런 고민 있으신가요?
섹션_설명: 반려견의 문제 행동으로 힘드셨다면, 전문가의 도움이 필요합니다.

문제_목록:
  - 아이콘: Volume2
    제목: 과도한 짖음
    설명: 초인종, 다른 강아지, 지나가는 사람만 보면 짖어요
  - 아이콘: AlertTriangle
    제목: 공격적인 행동
    설명: 물거나 으르렁거려서 다른 강아지와 어울리지 못해요
  - 아이콘: Heart
    제목: 분리불안
    설명: 혼자 두면 짖고, 물건을 망가뜨리고, 대소변 실수를 해요
```

## 9. CTA 섹션

**파일: `components/sections/CTA.tsx`**

상담 신청 폼 섹션입니다.

```yaml
제목: {예: 무료 상담 신청하기}
부제목: {예: 전문 상담사가 24시간 내 연락드립니다}

폼_필드:
  - 라벨: 이름
    필수: true
    placeholder: {예: 홍길동}
  - 라벨: 연락처
    필수: true
    placeholder: {예: 010-0000-0000}
  - 라벨: 이메일
    필수: false
    placeholder: {예: email@example.com}
  - 라벨: {커스텀 필드}
    필수: false
    placeholder: {예시}

성공_메시지: {예: 상담 신청이 완료되었습니다!}

특징:
  - {예: 무료 상담}
  - {예: 24시간 내 연락}
  - {예: 전문 상담사}
```

**현재 예시:**
```yaml
제목: 지금 바로 무료 상담 신청하세요
부제목: 전문 상담사가 24시간 내에 연락드립니다

폼_필드:
  - 라벨: 이름
    placeholder: 이름을 입력하세요
  - 라벨: 이메일
    placeholder: 이메일을 입력하세요
  - 라벨: 연락처
    placeholder: 연락처를 입력하세요
  - 라벨: 견종
    placeholder: 반려견의 견종을 입력하세요
  - 라벨: 문제 행동
    placeholder: 고민되는 문제 행동을 자유롭게 적어주세요

성공_메시지: 상담 신청이 완료되었습니다! 24시간 내에 연락드리겠습니다.

특징:
  - 무료 상담
  - 맞춤 프로그램
  - 평생 사후관리
```
